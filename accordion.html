Frequently Asked Questions

This is a modern, fully functional accordion built with pure HTML, CSS, and vanilla JavaScript. It features smooth animations, glassmorphism styling, and full keyboard accessibility.

  • No external libraries required
  • Responsive across all devices
  • Smooth open/close transitions

This accordion comes packed with modern UI features out of the box:

  • Glassmorphism design with frosted-glass effect
  • Smooth cubic-bezier animations for open and close
  • Animated chevron rotation indicator
  • Active state highlighting with purple glow
  • Keyboard navigation support (Enter & Space)
  • ARIA attributes for screen readers

Absolutely! The color scheme uses a purple-violet gradient palette that you can easily modify. The key color values to change are:

  • Background gradient on the body element
  • Accent color #7c60ff used for highlights
  • Glass panel background opacity and blur values
  • Border colors and glow shadow values

Yes, the accordion is fully responsive. It adapts to all screen sizes with adjusted padding, font sizes, and icon dimensions on smaller screens. The glassmorphism effect works beautifully on both desktop and mobile browsers.

The accordion uses a single-open mode by default, meaning only one panel can be expanded at a time. When you click a new panel, the previously open one closes automatically with a smooth animation. Clicking an already open panel will close it.

The transition uses max-height with a cubic-bezier easing for a natural, fluid motion.

Accessibility is built in from the ground up:

  • Semantic button elements for keyboard focus
  • aria-expanded attributes toggle dynamically
  • Full keyboard support with Enter and Space keys
  • Visible focus indicators for keyboard navigation
  • Proper contrast ratios for all text elements
